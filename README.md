# naokihaba blog

Personal blog built with Nuxt and deployed to Cloudflare Workers.

## Tech Stack

- [Nuxt](https://nuxt.com/)
- [@nuxt/content](https://content.nuxt.com/)
- [UnoCSS](https://unocss.dev/)
- [Cloudflare Workers](https://workers.cloudflare.com/)

## Commands

| Command | Action |
| :--- | :--- |
| `pnpm install` | Install dependencies |
| `pnpm dev` | Start local dev server at `localhost:3000` |
| `pnpm generate` | Generate static site to `.output/public/` |
| `pnpm deploy` | Generate and deploy to Cloudflare Workers |
